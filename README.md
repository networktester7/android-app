# android-app

The project is going to sync from available APIs to request the latest weather information from common websites. Once the application has downloaded the necessary data, it will display on the device’s screen. Users will be able to view the weather information of their location by searching for it from within the app. 

As mobile devices are becoming increasingly powerful, they have rapidly advanced the human landscape by having essential information be instantly searchable and accessible. One such requirement that remains essential is the need to check what the current temperature is outside to and plan your day and your routine accordingly. Having this information being displayed on your mobile device means that you are able to access it from anywhere you are as long as you have a network connection. 
The platform is going to be Android OS and will primarily be targeted for phones. 

The backend is going to consist of a weather API service that distributes the necessary information to the device. 

The functionality is going to consist of a display interface that shows the current weather information of where you selected, as well as a search function to allow you to enter a location you choose that you want to see the weather of.
